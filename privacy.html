Privacy Policy

Last updated: July 20, 2026

1. Who we are

DealRadar Publisher ("the Tool") is a personal content scheduling application operated by an individual developer ("the Operator") solely for the Operator's own TikTok account.

2. Data we collect

The Tool does not collect, store, or process personal data from any third party. The only data handled by the Tool is:

  • OAuth access tokens for the Operator's own TikTok account, stored locally on the Operator's machine;
  • Content files (images, videos, captions) created by the Operator for publication;
  • Post identifiers and basic performance metrics of the Operator's own posts, retrieved via TikTok's official APIs.

3. What we do not do

  • No data about TikTok users other than the Operator is collected or stored.
  • No data is sold, shared, or transferred to third parties.
  • No tracking, advertising, or analytics services are embedded in the Tool.
  • No cookies are used by this website.

4. Data storage and security

All data handled by the Tool is stored locally on the Operator's own computer. Access tokens are kept in local configuration files not shared with any external service.

5. Data deletion

TikTok authorization can be revoked at any time from the TikTok app (Settings → Security → Apps and websites), which invalidates the Tool's access. Locally stored tokens and data can be deleted by the Operator at any time.
